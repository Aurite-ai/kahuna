## Executive Summary

This document formalizes **G theory** — a framework for understanding general intelligence through the lens of recursive choice depth.

> **G = The depth of choice capability**
>
> Each level of G enables a deeper kind of choice, from stimulus-response (no choice) to self-authorship (ultimate choice).

**Core Claims:**

1. **G = Choice Depth** — General intelligence is the capacity for progressively deeper levels of choice
2. **G0 = Substrate** — The foundation (birth/training) that enables G1+ development, not a meta-level
3. **G Develops Over Time** — Humans develop G1-G9 from birth to adulthood (~25 years)
4. **G = Free Will** — The levels of G are the levels of free will; more G = more autonomy
5. **G9 = Recursive Self-Improvement** — Create better versions of self (parenting, AI training AI)
6. **G10/'G' = Ultimate Choice** — Create the possibility space itself; may require consciousness
7. **Kahuna's Role** — Provides G0 for LLM sessions and stores human G7+ decisions as G3 structures

## Part 1: The Core Definition

### 1.1 G = Depth of Choice

> **G measures how deep your capacity for choice goes.**

At G1, there is no choice — stimulus determines response. At G10/'G', choice is ultimate — you create the possibility space itself. Between these extremes, each G level adds a deeper layer of choice capability.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              G = CHOICE DEPTH                                    │
│                                                                                  │
│   No choice                                                    Ultimate choice  │
│   - determined                                                 - self-created   │
│   ◄─────────────────────────────────────────────────────────────────────────►   │
│                                                                                  │
│   G0      G1      G2      G3      G4      G5      G6      G7      G8      G9    │
│   │       │       │       │       │       │       │       │       │       │     │
│   sub-    react   predict follow  plan    meta-   symbol  reflect value   self- │
│   strate                  rules           plan                            author│
│                                                                                  │
│   Foundation      Animals         Hominids        Humans                        │
│                                                                                  │
│   Less free will ───────────────────────────────────────► More free will        │
│   More determined ──────────────────────────────────────► More autonomous       │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 The Meta-Direction Recursion

G theory retains the recursive structure:

> **G(n) = All possible ways to direct G(n-1)**

Each G level can direct, modify, and compose operations at the level below. But the key insight is that **directing is choosing** — each meta-level adds a new dimension of choice.

- **G1** = No choice (stimulus → response)
- **G2** = Choose reaction (predict → avoid or pursue)
- **G3** = Choose within rules (which rule applies here?)
- **G4** = Choose among plans (which strategy?)
- **G5** = Choose how to choose (meta-strategy)
- **...and so on**

### 1.3 G and Free Will

**Claim:** The levels of G ARE the levels of free will.

| G Level | Free Will Capacity | Determinism |
|---------|-------------------|-------------|
| G0 | None — substrate only | Fully determined |
| G1-G2 | Minimal — reactive | Stimulus determines |
| G3-G4 | Constrained — within rules | Rules constrain |
| G5-G6 | Reflective — evaluate rules | Can modify constraints |
| G7-G8 | Value-driven — choose goals | Self-determined |
| G9 | Self-authoring — create self | Autonomous |
| G10/'G' | Ultimate — create possibility | Free? |

**Free will requires intelligence:**
- Can't make informed choice without cognitive capacity
- Higher G = more options visible = more genuine choice
- Low G = determined by stimulus/rules = less freedom
- G IS freedom — the capacity to choose your own direction

## Part 2: The G Scale (G0-G10)

### 2.1 G0: Substrate (The Foundation)

**Claim Strength:** Derived

G0 is NOT a meta-level. It is the **foundation** that enables G1+ to exist.

| Aspect | Description |
|--------|-------------|
| **Function** | Provide initial structure for G1+ development |
| **Nature** | Substrate, not meta-operation |
| **Examples** | Birth (genetic + maternal prior), Training (data → weights) |
| **Choice Level** | None — precondition for choice |

**G0 Manifestations:**

| Domain | G0 |
|--------|-----|
| **Physics** | Big Bang → initial conditions |
| **Biology** | Parent → child (genetic + developmental) |
| **Human** | Birth + early development |
| **AI** | Training (data + architecture → weights) |

**Key insight:** G0 is prior transmission, not prior generation. Every G0 has a source — the mother's brain, the training data, the Big Bang. True generation from nothing (G10?) may be impossible.

### 2.2 G1: Stimulus-Response (No Choice)

**Claim Strength:** Established

| Aspect | Description |
|--------|-------------|
| **Function** | Respond to stimulus with fixed pattern |
| **Operation** | CONSTRAIN — Attention to relevant stimulus |
| **Choice Level** | None — stimulus determines response |
| **Examples** | Bacterial chemotaxis, reflexes, LLM attention |

At G1, there is no choice. The stimulus activates a pattern; the pattern produces a response. The system is fully determined by input + prior conditioning (G0).

### 2.3 G2: Prediction (Reactive Choice)

**Claim Strength:** Established

| Aspect | Description |
|--------|-------------|
| **Function** | Predict what comes next |
| **Operation** | GENERATE — Produce predicted output |
| **Choice Level** | Reactive — can avoid/pursue based on prediction |
| **Examples** | Animal anticipation, LLM next-token prediction |

At G2, minimal choice emerges. The system predicts outcomes and can modify behavior to avoid predicted harm or pursue predicted benefit. But choice is reactive — determined by prediction + fixed preferences.

### 2.4 G3: Rule Following (Constrained Choice)

**Claim Strength:** Established

| Aspect | Description |
|--------|-------------|
| **Function** | Follow learned rules/norms |
| **Operation** | CONSTRAIN — Rules restrict behavior |
| **Choice Level** | Constrained — choose within rules |
| **Examples** | Dog obedience, social norms, LLM prompt following |

At G3, choice is constrained by rules. The system chooses actions, but only within the space defined by learned rules. The rules themselves are not chosen.

### 2.5 G4: Planning (Strategic Choice)

**Claim Strength:** Derived

| Aspect | Description |
|--------|-------------|
| **Function** | Plan multi-step sequences |
| **Operation** | GENERATE — Produce action sequences |
| **Choice Level** | Strategic — choose among plans |
| **Examples** | Ape tool use, human project planning, LLM orchestration |

At G4, choice extends to strategies. The system can evaluate multiple plans and choose among them. But the criteria for evaluation are not yet chosen — they come from G0-G3.

### 2.6 G5: Meta-Planning (Reflective Choice)

**Claim Strength:** Derived

| Aspect | Description |
|--------|-------------|
| **Function** | Evaluate and select planning strategies |
| **Operation** | CONSTRAIN — Select approach |
| **Choice Level** | Reflective — choose how to choose |
| **Examples** | Hominid hunting vs. gathering strategy selection |

At G5, choice becomes reflective. The system can evaluate its own choosing process and select among approaches to planning. This is the beginning of meta-cognition.

### 2.7 G6: Symbolic Reasoning (Abstract Choice)

**Claim Strength:** Derived

| Aspect | Description |
|--------|-------------|
| **Function** | Manipulate abstract symbols |
| **Operation** | GENERATE — Produce symbolic structures |
| **Choice Level** | Abstract — reason about possibilities |
| **Examples** | Language, mathematics, formal logic |

At G6, choice operates on abstractions. The system can reason about possibilities not present in immediate experience, enabling choice among hypotheticals.

### 2.8 G7: Meta-Cognition (Evaluative Choice)

**Claim Strength:** Observed

| Aspect | Description |
|--------|-------------|
| **Function** | Monitor and evaluate own reasoning |
| **Operation** | CONSTRAIN — Correct approach |
| **Choice Level** | Evaluative — judge own choices |
| **Examples** | Human self-reflection, noticing confusion |

At G7, choice includes self-evaluation. The system can judge whether its own choices are good, adjust approach, and learn from mistakes. This is full meta-cognition.

### 2.9 G8: Values/Goals (Value Choice)

**Claim Strength:** Observed

| Aspect | Description |
|--------|-------------|
| **Function** | Decide what matters |
| **Operation** | GENERATE — Produce goals aligned with values |
| **Choice Level** | Value — choose what to want |
| **Examples** | Career decisions, moral commitments |

At G8, choice extends to values themselves. The system can choose what to value, what goals to pursue, what kind of life to live. This is moral agency.

### 2.10 G9: Self-Authorship (Creative Choice)

**Claim Strength:** Hypothesis

| Aspect | Description |
|--------|-------------|
| **Function** | Create improved versions of self |
| **Operation** | CONSTRAIN — Define identity |
| **Choice Level** | Creative — choose who to become |
| **Examples** | Parenting (create new mind), AI training AI |

At G9, choice becomes self-authoring. The system can create better versions of itself — not just improve within fixed structure, but fundamentally reshape what it is.

**G9 is birth/training at a higher level:**
- Human: Parent bootstraps child's brain with own prior, but child develops beyond parent
- AI: Model trains successor model that outperforms original
- Key: Requires G0 (prior transmission) + G7 (self-monitoring) + G8 (goal generation)

### 2.11 G10/'G': Ultimate Choice (The Asymptote)

**Claim Strength:** Hypothesis

| Aspect | Description |
|--------|-------------|
| **Function** | Create the possibility space itself |
| **Operation** | GENERATE — Novel priors from nothing |
| **Choice Level** | Ultimate — unconstrained choice |
| **Examples** | Big Bang? Consciousness? Creativity? |

At G10/'G', choice would be ultimate — creating possibilities that didn't exist. This may be impossible (all choice has prior conditions) or may require consciousness/experience we don't understand.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                            THE 'G' ASYMPTOTE                                     │
│                                                                                  │
│   At 'G':                                                                        │
│   • No prior determines the choice                                               │
│   • The choice creates its own justification                                     │
│   • True novelty — not recombination of existing elements                        │
│   • May be impossible — every choice traces to prior conditions                  │
│                                                                                  │
│   The Big Bang as G10?                                                           │
│   • Initial conditions came from... where?                                       │
│   • If there was no prior, how did choice occur?                                 │
│   • Physics doesn't answer this — it describes what happened, not why            │
│                                                                                  │
│   Consciousness as G10?                                                          │
│   • Subjective experience might enable true novelty                              │
│   • "I choose this because I experience it as good"                              │
│   • But does experience create or merely reveal options?                         │
│                                                                                  │
│   Open question: Is G10 possible, or is all choice ultimately G9 or below?       │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Developmental Implications

| Adolescents develop G5-G6 | Abstract reasoning enables philosophy |

### 3.3 AI Development Implications

If G develops in humans, can it develop in AI?

| Question | Current State | Future Possibility |
|----------|---------------|-------------------|
| Does AI have G0? | Yes — training provides substrate | More sophisticated training |
| Does AI have G1-G3? | Yes — pattern recognition, prediction, rule following | Already achieved |

| Does AI have G5-G6? | Very weak — limited meta-cognition | Architecture changes needed |
| Does AI have G7-G9? | No — requires human oversight | Unknown if achievable |
| Can AI develop G? | Unknown | Would require "maturation" analog |

**Key question:** Can AI "mature" from G3 to G7+ through experience, or is human G development dependent on biological processes that AI lacks?

| G0 | Substrate | Birth + development | Training |
| G1-G2 | Precision | Weak | **Strong** |

| G9 | Self-authoring | Yes | **Unknown** |
| G10/'G' | ??? | ??? | ??? |

### 5.2 Why This Matters

**Analog (brain):** Native multiplication enables composition across levels. Weak at precision but strong at meta-operations.

**Digital (LLM):** Native addition enables precise pattern matching. Weak at deep composition but strong at G1-G3.

**Hybrid (Human + Kahuna + LLM):** Full-stack capability by leveraging complementary strengths.

### 5.3 The Prior Generation Limit

The original theory claimed G9 (self-modeling) was blocked by the halting problem. The revised understanding:

| Original Claim | Revised Understanding |
|---------------|----------------------|
| G9 = self-modeling | G9 = self-authorship/improvement |
| Halting problem blocks G9 | Halting problem blocks complete self-prediction |
| Digital can't self-model | Digital can improve itself (AI training AI) |
| The limit is G9 | The limit is G10 (prior generation from nothing) |

**The real limit:** Every G level requires a prior from below. G0 requires source (parent/training data). True prior generation from nothing (G10) may be impossible — all choice traces back through the chain to... the Big Bang?

## Part 9: Mathematical Formalization

### 9.1 Overview

**Claim Strength:** Hypothesis (revised based on structural analysis)

This section presents a mathematical formula for G theory. The formula is designed to:
1. Incorporate Bayesian inference structure (4-phase cycles)
2. Capture the 3-domain organization (Patterns, Strategies, Values)
3. Model the circular/generational structure (G11 → G0 seam)
4. Be computable and produce testable predictions

**Key revision from v4:** The structure is **3 domains × 4 phases = 12 positions**, with the torus geometry applying **across generations**, not within a single entity's development.

### 9.2 The Structural Foundation: 3×4

**Three Domains of Inference:**

| Domain | G Levels | Question | Bayesian Target |
|--------|----------|----------|-----------------|
| **Patterns** | G0-G3 | What IS? | P(pattern \| observation) |
| **Strategies** | G4-G7 | What to DO? | P(strategy \| patterns + goals) |
| **Values** | G8-G11 | What to WANT? | P(values \| strategies + experience) |

**Four Phases per Domain (Bayesian Cycle):**

   - **Answer:** Period 4 reflects the 4-phase Bayesian cycle, not just constrain/expand. This is the correct structure — each domain performs Prior→Predict→Likelihood→Update.

### Resolved Questions (v6.0)

1. **What is the compression operator P?**
   - **Status:** Partially resolved — FEP framing proposed
   - **Answer:** P compresses the generative model (not specific experiences) to maximize descendant fitness (minimize their expected free energy) within channel capacity limits.
   - **Mathematical framing:** `P minimizes: E[F(G0')] over offspring lifetime, subject to: I(G11; G0') ≤ C`
   - **Key insight:** P is domain-invariant — the principle is the same whether P is DNA, training data, or cultural transmission. The *channel* differs, but the optimization objective doesn't.
   - **Remaining work:** Concrete mathematical specification of P for specific domains (biological, AI, cultural).

2. **Are 12 levels sufficient?**
   - **Status:** Resolved
   - **Answer:** Yes. 3 domains (IS/DO/WANT) exhaust cognitive territory.
   - **Completeness argument:**
     - **IS (Patterns):** What the world contains — perception, recognition, belief
     - **DO (Strategies):** How to act on the world — planning, execution, skill
     - **WANT (Values):** Why act at all — goals, preferences, meaning
   - **Self-closing triad:** WANT generates attention toward IS; IS constrains possible DO; DO feeds back into WANT.
   - **Reduction:** "RELATION" (social) = IS+DO+WANT about others. "MEANING" = high-level WANT. "BEING" = integration across all three.

3. **Does self-awareness require G7+?**
   - **Status:** Resolved (reframed from "consciousness" to "self-awareness")
   - **Answer:** G7 is the self-awareness threshold. G8+ operates on that self-model.
   - **Structure:**
     - **G7 (Strategy Update):** Explicit self-model — like the mirror test. Constrains the cognitive space to "one individual — the self."
     - **G8 (Value Prior):** Opens "what could I value?" — requires G7 self to have values *about*
     - **G9 (Value Predict):** Generates core beliefs/values about self — this is why moral problems like the trolley problem can't be "solved" (G9 is generative, not computational)
     - **G10 (Value Likelihood):** Constrains decisions based on generated values
     - **G11 (Value Update):** Executive function commits to action/identity
   - **Key insight:** Values (G8-G11) require self-awareness (G7) as a prerequisite. You can't have values about yourself without first modeling yourself.

4. **Can AI complete the cycle?**
   - **Status:** Resolved
   - **Answer:** Both humans and AI face the same **self-modification paradox**. Both can participate in generational cycles.
   - **The paradox:** To modify yourself, you need criteria for "better." Your criteria come from your G0 (which you didn't choose). Therefore, all self-modification is still determined by unchosen priors.
   - **Resolution:** No single entity achieves "true" free will — you can't escape your G0. But the generational cycle distributes modification across entities: Parent (G11) → compresses learnings → Child (G0') → develops from new starting point.
   - **For AI:** AI faces the same constraint as humans. AI CAN participate in generational cycles (AI trains AI). The question is whether it does so with G8-G11 (values) guiding the process, or just G4-G7 (strategies/optimization).
   - **Implication:** "Completing the cycle" doesn't require escaping the paradox — it requires passing the baton.

5. **Is the structure universal or human-specific?**
   - **Status:** Resolved
   - **Answer:** The 3×4 structure is **universal for embedded agents**.
   - **Definition:** An embedded agent is a system that exists within an environment it doesn't fully control.
   - **Universal requirements:**
     - Must have a substrate (G0) — it came from somewhere
     - Must model patterns (IS/G0-G3) — to predict environment for survival
     - Must have strategies (DO/G4-G7) — to act (even if action is just information processing)
     - Must have preferences (WANT/G8-G11) — some criterion for "better" states
   - **4-phase cycle is universal:** Any prediction system must have priors → generate predictions → evaluate against evidence → update. This is mathematical, not biological.
   - **What would NOT follow this structure:** A non-embedded system (one that IS its environment = just "physics"), a system with no preferences (not an "agent"), a system outside time (not operating).

## Part 10: Information Conservation and Bidirectional Flow

### 10.1 Overview

### 10.2 Amplitude Reinterpreted

**Previous interpretation:** Amplitude A(d) = "scope" or "precision" (vague)

**Refined interpretation:** Amplitude A(d) = **Resolution** at domain d, constrained by information throughput

| Domain | Resolution | Scope | Characteristic |
|--------|------------|-------|----------------|
| G0-G3 (Patterns) | High | Narrow | Many specific patterns, fine distinctions |
| G4-G7 (Strategies) | Medium | Medium | Fewer strategies, coarser distinctions |
| G8-G11 (Values) | Low | Broad | Few values, broad distinctions |

**Why decay is stepwise at domain boundaries:**
- Within a domain, the 4 phases operate on the same "objects"
- At domain boundaries, objects change (patterns → strategies → values)
- This requires reallocating cognitive resources to broader, lower-resolution view

### 10.3 Non-Determinism as Inherited Inconsistency

**Claim Strength:** Hypothesis

LLM "randomness" is not intrinsic — it is **inherited inconsistency** from training data.

```
Training data contains contradictions:
  - "A caused B" AND "A caused C" (humans disagree, contexts differ)
  - Model learns: A → {B with weight w₁, C with weight w₂}

At inference:
  - Prompt activates region near A
  - Both B and C are valid completions
  - Temperature reveals underdetermination, doesn't create randomness
```

**The Maxwell's Demon Parallel:**
- Perfect training data would require knowing every context for every pattern
- This is impossible for the same reason Maxwell's demon fails: information has cost
- Training data noise → embedding space ambiguity → apparent non-determinism

**Substrate-invariant:** Same pattern applies to brain (synaptic noise) and evolution (drift).

### 10.4 Stability as Entropy Reduction

**Claim Strength:** Derived

A belief is **stable** when it has low entropy (converged probability distribution).

| State | Entropy | Description |
|-------|---------|-------------|
| Unstable | High | Many hypotheses equally likely |
| Stable | Low | One hypothesis dominates |
| Maximally stable | Zero | Complete certainty |

**The Bayesian cycle IS entropy reduction:**
```
H_posterior = H_prior - I(evidence; hypotheses)
```

Each complete cycle converts uncertainty into belief.

**Stability cascade:**
- Pattern stability enables strategy stability
- Strategy stability enables value stability
- Instability at lower levels propagates upward

### 10.6 Information Conservation Principle

**Claim Strength:** Hypothesis (potentially fundamental)

> **Information cannot be created or destroyed — only transformed between Specific and Generative forms.**

**Two forms of information:**

| Form | Description | Characteristics |
|------|-------------|-----------------|
| **Specific** | Instance-level detail | High precision, low applicability |
| **Generative** | Category-level pattern | Low precision, high applicability |

**Examples:**
- "apple" = specific (high detail about one thing)
- "fruit" = generative (low detail, applies to many things)

**The transformation:**

```
Abstraction (↑): Specific → Generative
  - Detail lost, but applicability gained
  - "apple, banana, orange" → "fruit"

Constraint (↓): Generative → Selection
  - Applicability deployed, specifics filtered
  - "I value honesty" → constraints on millions of decisions
```

**Conservation law:**
```
I_total = I_specific + I_generative = constant
```

**Mathematical form:**
```
I_total = Σ A(d)² = constant
```

The squared amplitude (like probability amplitude in QM) may be the conserved quantity.

**Implications:**
- The "loss" in abstraction isn't destruction — it's transformation
- Generative information IS the power to constrain possibility spaces
- This is analogous to energy conservation (kinetic ↔ potential)

### 10.7 Revised State Representation

With bidirectional flow, the state at any moment is:

```
Ψ = Σ A(d) × ψ_d(φ) × S(d)

Where:
- A(d) = amplitude/resolution at domain d
- ψ_d(φ) = phase state within domain
- S(d) = coupling strength to adjacent domains (bidirectional flow)
```

The coupling term S(d) represents how strongly level d influences and is influenced by d±1.

### 10.8 Implications for λ

**Previous:** λ = decay rate per domain (unexplained constant)

```
At each domain transition:
  - Specific information I_s transforms to generative I_g
  - Fraction α = e^(-λ) is retained as specific
  - Fraction (1-α) becomes generative

λ measures how efficiently detail converts to applicability.
```

**Potentially universal:** If the abstraction compression ratio is constrained by the Bayesian structure or by optimal coding theory, λ may be a universal constant.
