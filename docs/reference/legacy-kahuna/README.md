# Legacy Kahuna (Kahuna 1.0) Reference

This directory contains analysis of the **original Kahuna codebase** (Kahuna 1.0).

## Purpose

These documents capture patterns, tech stack decisions, and learnings from the first version of Kahuna. They are useful for:

- Understanding patterns to avoid (see [SUMMARY.md](./SUMMARY.md))
- Providing context for rebuild decisions
- Reference when migrating features

| [frontend-patterns.md](./frontend-patterns.md) | Frontend architecture patterns |
| [tech-stack.md](./tech-stack.md)               | Technology stack analysis      |

## Important Note

This is historical reference material. The Kahuna 2.0 repository is a **complete rebuild**, not a refactor. These patterns describe what existed, not what we're building.
