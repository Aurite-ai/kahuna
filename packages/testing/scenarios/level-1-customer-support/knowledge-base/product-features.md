# TechFlow Pro - Product Features

## Core Features

### Project Management
Create and manage projects with customizable workflows. Each project includes:
- **Task lists**: Organize work into lists (e.g., To Do, In Progress, Done)
- **Kanban boards**: Drag-and-drop cards for visual workflow management
- **Gantt charts**: Timeline view for planning and dependencies (Pro+)
- **Calendar view**: See tasks by due date

### Task Management
Every task in TechFlow includes:
- Title and description (with rich text formatting)
- Assignee(s) - assign to one or multiple team members
- Due date and time
- Priority level (Low, Medium, High, Urgent)
- Labels/tags for categorization
- Subtasks for breaking down complex work
- Comments for discussion
- File attachments (up to 100 MB per file)
- Activity log tracking all changes

### Team Collaboration
- **Real-time updates**: See changes instantly without refreshing
- **@mentions**: Tag teammates in comments to notify them
- **Shared views**: Create saved filters that the whole team can use
- **Guest access**: Invite external collaborators with limited permissions (Starter+)

### Time Tracking
Built-in time tracking for tasks:
- Start/stop timer with one click
- Manual time entry
- Time reports by project, task, or team member
- Export time data for invoicing

## Advanced Features (Pro & Enterprise)

### Custom Fields
Create custom data fields for tasks:
- Text fields
- Number fields
- Date fields
- Dropdown menus (single or multi-select)
- Checkbox fields
- Currency fields

### Automations
Automate repetitive work with rules:
- **Triggers**: Task created, status changed, due date approaching, etc.
- **Conditions**: Filter by assignee, project, priority, custom fields
- **Actions**: Move task, assign user, send notification, update field, create task

Example automations:
- When a task is marked "Done", notify the project manager
- When a due date is missed, increase priority to "Urgent"
- When a task is assigned, add it to the assignee's "My Tasks" view

### Advanced Reporting
Generate insights with customizable reports:
- **Workload view**: See team capacity and balance assignments
- **Burndown charts**: Track progress toward deadlines
- **Velocity tracking**: Measure team productivity over time
- **Custom dashboards**: Build your own charts and metrics

### Dependencies
Link related tasks:
- **Blocking**: Task A must finish before Task B can start
- **Related**: Tasks are connected but not dependent
- **Duplicate**: Tasks are copies of each other

### Templates
Save time with reusable templates:
- **Project templates**: Recreate entire project structures
- **Task templates**: Standardize recurring tasks
- **Checklist templates**: Reusable subtask lists
