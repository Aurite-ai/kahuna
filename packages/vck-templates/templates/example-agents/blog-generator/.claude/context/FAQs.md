# AI News Blog Post Generator - Requirements

## News Gathering

**Q: How should the agent gather AI news?**
A: Use whatever free news API is available

**Q: Are there specific AI news sources you prefer?**
A: Focus on TechCrunch AI as the primary source

## Content Generation

**Q: What kind of content should be included in the blog posts?**
A: Short summaries of articles with overarching themes

**Q: What is the desired length and depth of the blog posts?**
A: Target length: 500-1000 words

**Q: Which LLM should the agent use for content generation?**
A: Use Anthropic's Claude for content generation

**Q: What types of SVG diagrams/infographics should be generated?**
A: Use whatever diagrams are appropriate for the content

**Q: How should the agent determine what to visualize in each post?**
A: Automatic detection - the agent should determine what to visualize

A: Run on-demand (not scheduled)

A: No special markdown structure, just standard markdown

## Quality & Safety
