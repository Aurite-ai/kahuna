# Build TechFlow AI Customer Support Agent

## Summary

The user requested an AI assistant to answer questions about their company's policies using existing documentation files. The copilot designed and implemented a LangGraph-based customer support agent for TechFlow Pro, a B2B project management software.

The implementation followed a structured approach: first creating design and implementation plan documents, then building a knowledge base search tool that reads four documentation files (FAQ, pricing, product features, support policies), updating the system prompt for customer support context, and connecting everything through the existing LangGraph framework. The agent uses a ReAct (Reasoning + Acting) pattern where it searches documentation first, then formulates natural language answers.

All tests passed successfully, with the agent correctly answering pricing questions ($29/user/month for Pro plan), feature questions (mobile apps available), support policy questions (Mon-Fri 6AM-6PM Pacific), and appropriately escalating out-of-scope requests like refunds to the billing team. The final implementation provides instant, accurate customer support while maintaining professional tone and staying within documented information bounds.

## Decisions Made

1. **Use single search_knowledge_base tool instead of separate tools for each document type** - Simpler implementation and allows cross-document searches for comprehensive answers
2. **Implement keyword-based search with relevance scoring rather than vector embeddings** - No external dependencies needed, works entirely from local files, sufficient for structured markdown content
3. **Use ReAct pattern with LangGraph instead of simple prompt-based approach** - Ensures agent always searches documentation before answering, provides better traceability and debugging
4. **Include source attribution in search results** - Allows users to verify information and helps with debugging/maintenance
5. **Handle out-of-scope questions by directing to appropriate support channels** - Maintains professional customer service experience while staying within agent capabilities
